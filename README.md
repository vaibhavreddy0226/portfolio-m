# Vaibhav Reddy — Portfolio

Personal portfolio website built with vanilla HTML and CSS. Showcases my projects, skills, and contact information.

🔗 **Live Site:** [vaibhavreddy0226.github.io/portfolio-m](https://vaibhavreddy0226.github.io/portfolio-m/)

---

## Project Overview

A fully static, multi-page portfolio site with no frameworks or build tools. It features a dark theme with a custom CSS design system, smooth hover interactions, a spinning ring animation on the hero photo, responsive layouts for all screen sizes, and a contact form. Everything is driven by plain HTML and a single CSS file using custom properties.

---

## Setup Instructions

Make sure you have the following before running locally:

- A modern browser (Chrome, Firefox, Edge, Safari)
- [Git](https://git-scm.com/) installed
- Optionally: [VS Code](https://code.visualstudio.com/) with the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension for auto-reload

Clone the repository:

```bash
git clone https://github.com/vaibhavreddy0226/portfolio-m.git
cd portfolio-m
```

---

## How to Run

**Option 1 — Open directly in browser:**

```bash
open index.html
```

Or double-click `index.html` in your file explorer. No server required since there is no backend.

**Option 2 — VS Code Live Server (recommended):**

1. Open the project folder in VS Code
2. Right-click `index.html`
3. Select **Open with Live Server**

The site will open at `http://127.0.0.1:5500` and auto-reload on any file save.

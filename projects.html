02

Hospital Delivery Robot Simulation

Designed and simulated an autonomous delivery robot for dynamic hospital environments using MATLAB. The system integrates A* global path planning with a proportional motion controller and a hybrid obstacle-handling strategy — waiting for temporary blockages and executing local reroutes around permanent ones. Built on a 150×150 m hospital floor plan with 10 rooms, covering over 110 m at an average speed of 0.75 m/s while handling real-time obstacles without collision.

03

Author-Based Book Finder

Developed a lightweight web application that allows users to search for an author and retrieve up to 10 of their published books using the Open Library API. The application dynamically displays book titles along with their cover images, providing a simple and interactive way to explore an author’s work. Built using HTML, CSS, and JavaScript, the project focuses on API integration, real-time data fetching, and responsive UI design.
